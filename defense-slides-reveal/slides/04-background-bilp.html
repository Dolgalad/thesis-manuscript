max\(x\) \(c^\top x\)
\(\text{s.t.}\) \(Ax \leq b\)
\(x \in \{0,1\}^n\)
Decision variables \(x_i \in \{0,1\}\)
Cost function \(c^\top x\)
Constraints \(Ax \leq b\) couples variables. \(A \in \mathbb{R}^{m \times n},\quad b \in \mathbb{R}^m\)
Feasible set \(\mathcal X = \{x \in \{0,1\}^n : Ax \leq b\}\)
This slide introduces the basic structured prediction view of binary ILPs. The BILP defines binary variables, a linear objective, and linear constraints. The constraints define a feasible set, so the prediction is a complete binary assignment rather than independent labels.
