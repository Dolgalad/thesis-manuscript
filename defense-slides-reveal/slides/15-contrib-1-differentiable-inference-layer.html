Approximate inference becomes a differentiable layer between predicted MRF parameters and the training objective.
The key message is that LBP or NMF is not only used after training. We run a fixed number of inference iterations inside the model, treat the iterations as differentiable computations, and backpropagate through them. This allows the parameter predictor to be trained for the marginals produced by approximate inference, rather than only for raw local potentials.
