This slide should be read as a hierarchy of comparisons. First, the factorized DGN baseline gives independent variable scores and is decoded with EGG. EGG is the no-reinference decoder: it uses one fixed set of scores greedily. The canonical MRF, even though hand-crafted, already improves substantially because feasibility is represented inside the inference model. The learned MRF improves further, especially with LBP and Guided Decimation. GD repeatedly conditions on the partial solution and recomputes beliefs before making the next decision. The main message is that learned MRFs improve solution quality, and reinference through guided decimation makes the learned marginals more useful during decoding.
