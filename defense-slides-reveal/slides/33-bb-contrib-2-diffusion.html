Diffusion-style training signal

Learn to reverse a controlled corruption process

A clean target point is gradually corrupted as time increases. The model learns the reverse direction: starting from noise, it iteratively predicts a cleaner point.

Why MRFs? Corrupted data + time gives a partial view of the target sample; an MRF can better capture the remaining structure. The model should not correct each variable independently: inference can propagate constraint-aware information.
This version avoids showing an explicit assignment. The forward process is simply illustrated as a target point undergoing a random walk as time increases. The reverse process starts from a noisy point and repeatedly applies a neural denoising model. The important message is that the corrupted point and the time provide only a partial view of the original sample. An MRF layer can exploit structure to infer the missing information in a constraint-aware way.
