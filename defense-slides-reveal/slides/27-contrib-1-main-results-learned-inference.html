How to read the comparison

Factorized baseline DGN + EGG

Uses independent variable scores; fast, but weakest quality.

Canonical MRF CMRF + GD

Hand-crafted feasibility potentials already give strong structured marginals.

Learned MRF LMRF-LBP + GD

Learned factor potentials + conditional marginals give the best gaps.

Inference matters LBP vs NMF

NMF is competitive on Set Cover, but LBP is more robust on MIS.

The gain does not come only from better neural scores: it comes from using structured marginals and recomputing them during decoding.
This slide should be read as a hierarchy of comparisons. First, the factorized DGN baseline gives independent variable scores and is decoded with EGG. The canonical MRF, even though hand-crafted, already improves substantially because feasibility is represented inside the inference model. The learned MRF improves further, especially with LBP and Guided Decimation. EGG uses one set of marginals greedily, while GD repeatedly conditions on the partial solution and recomputes beliefs. The NMF variant performs well on Set Cover, but it is less robust on MIS, especially on the medium MIS instances where LMRF-NMF with GD degrades.
