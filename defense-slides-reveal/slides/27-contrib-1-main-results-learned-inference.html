How to read the comparison

Factorized baseline DGN + EGG

Uses independent variable scores; fast, but weakest quality.

Canonical MRF CMRF + GD

Hand-crafted feasibility potentials already give strong structured marginals.

Learned MRF LMRF-LBP + GD

Learned factor potentials + conditional marginals give the best gaps.

Inference matters LBP + GD

Recomputing LBP marginals during decimation is the most reliable decoding strategy.

This slide should be read as a hierarchy of comparisons. First, the factorized DGN baseline gives independent variable scores and is decoded with EGG. The canonical MRF, even though hand-crafted, already improves substantially because feasibility is represented inside the inference model. The learned MRF improves further, especially with LBP and Guided Decimation. EGG uses one set of marginals greedily, while GD repeatedly conditions on the partial solution and recomputes beliefs. This version focuses the comparison on the LBP-based inference variants and the effect of Guided Decimation.
