Guided decimation uses reinference to make marginals compatible with the partial solution, instead of reusing scores from the initial global mixture.
This example is designed so that the initial marginals are homogeneous. There are three symmetric optimal covers of size two: S1 with S2, S3 with S4, and S5 with S6. Before any decision, each set can belong to an optimal cover, so the marginal scores are identical. The decoder breaks symmetry by fixing x1, corresponding to selecting S1. Once S1 is fixed, the remaining uncovered elements are d, e, and f. Reinference conditions the MRF on the partial assignment A={x1=1}. The conditional marginal of x2 increases because S2 covers exactly the residual uncovered elements. This illustrates the purpose of guided decimation: each new decision should be made in the context of the partial solution already built.
