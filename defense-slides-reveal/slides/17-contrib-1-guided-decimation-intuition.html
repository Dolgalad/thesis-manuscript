Set Cover example

Several covers are initially symmetric

Initially, each set belongs to one optimal cover of size two: \(\{S_1,S_2\}\), \(\{S_3,S_4\}\), or \(\{S_5,S_6\}\).

Guided decimation uses reinference to make marginals compatible with the partial solution, instead of reusing scores from the initial global mixture.
This example is designed so that the initial marginals are homogeneous. There are three symmetric optimal covers of size two: S1 with S2, S3 with S4, and S5 with S6. Before any decision, each set can belong to an optimal cover, so the marginal scores are similar. Once we commit to S1, the remaining uncovered elements are d, e, and f. If we keep using the original marginals, the decoder has no reason to prefer the compatible completion S2 over the other sets. Reinference conditions the MRF on the partial assignment S1 equals one. The conditional marginal of S2 increases because S2 covers exactly the residual uncovered elements. This is the key purpose of guided decimation: each new decision should be made in the context of the partial solution already built.
