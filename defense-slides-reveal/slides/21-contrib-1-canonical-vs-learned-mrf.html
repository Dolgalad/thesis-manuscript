The scopes are still induced by the BILP, but the log-potential tables are predicted from GNN embeddings.

The important contrast is not the input graph: both models see the same BILP and use the same factor scopes. The canonical MRF uses fixed parameters: unary terms encode the objective and constraint factors encode feasibility by assigning zero to locally feasible assignments and a large negative value to infeasible ones. The learned MRF replaces this hand-designed construction by learned parameter heads applied to GNN embeddings.
