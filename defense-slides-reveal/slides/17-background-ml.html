Supervised setting
\[ \mathcal{D} = \{(x^{(k)}, y^{(k)})\}_{k=1}^{N} \]
Each training example contains an input \(x^{(k)}\) and a known target label \(y^{(k)}\).
\[ \hat y^{(k)} = f_\theta(x^{(k)}) \]
\[ \min_\theta \frac{1}{N} \sum_{k=1}^{N} L\!\left(f_\theta(x^{(k)}), y^{(k)}\right) \]
\[ L(\hat p, y) = -\sum_{c=1}^{C} \mathbf{1}[y=c]\log \hat p_c \]
For classification, the loss penalizes assigning low probability to the correct class.
In our case: the network will not directly output labels → it predicts parameters for an inference model
This slide introduces supervised training. A neural network maps an input example to a prediction. The loss function measures how different the prediction is from the target label. Training adjusts the parameters of the network to reduce the average loss over the dataset. For classification, cross-entropy is the standard loss: it penalizes the model when it assigns low probability to the correct class. This prepares the transition to our setting, where the network predicts parameters of an MRF and the loss is applied after approximate inference.
