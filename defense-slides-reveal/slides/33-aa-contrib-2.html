MRF-Augmented Denoising

From one-shot prediction to learned refinement dynamics

This section introduces the Chapter 5 denoising contribution. The central idea is to move from a single prediction pass to an iterative refinement process. A noisy or corrupted binary assignment is progressively denoised, and the denoising model can use an MRF inference layer to propagate constraint-aware information before decoding.

Denoising MRF

Predict + infer

Predict state-conditioned factors, then run LBP or NMF for marginals.

Training loss \(\mathcal{L}_{\mathrm{denoise}} = \mathbb{E}_{x^\star,t,\tilde{x}_t}\big[\ell(D_\omega(P,\tilde{x}_t,t),x^\star)\big]\) At test time: unroll denoising from a random binary initialization, then decode.
The denoising setup samples a clean solution, corrupts it according to a noise level, and trains the model to reconstruct the clean solution. The MRF-augmented versions differ from the direct DDGN baseline because their predictions pass through LBP or NMF before decoding. This makes denoising a structured refinement process rather than independent variable correction.

Problem-dependent behavior

Set Cover LBP-based denoising produces feasible samples frequently and low-gap decoded solutions.
MIS Raw feasibility remains 0%; GD improves DLBP to about 7.0%, but decoding is brittle.
The table in the manuscript compares DDGN, DLBP and DNMF. For Set Cover, DLBP clearly dominates the direct denoising baseline and the NMF variant. On SC-S it reaches 88.867 percent raw feasibility and a GD gap of 0.680 percent. On SC-M it reaches 84.615 percent raw feasibility and a GD gap of -0.233 percent. For MIS-S, raw feasibility is still zero for all denoising models, but GD reduces the DLBP gap to about 7.015 percent. The main message is that MRF-augmented denoising is useful, but the interaction with decoding depends on the problem structure.

Takeaway

Refinement works best when it remains structured

Chapter 5 changes the prediction view: solution generation becomes a trajectory, not a one-shot score vector.

01

Condition on the current state

The model sees the noisy assignment and the refinement time.

02

Use MRF inference inside refinement

LBP or NMF transforms local predictions into structured marginals.

03

Decode with feasibility in mind

Final quality still depends on the interaction between refinement and decoding.

Strong on covering problems; more fragile on packing problems with hard exclusion constraints.
The final takeaway is balanced. MRF-augmented denoising supports the thesis theme: independent variable scores are not enough, and structured inference improves the learned signal. However, refinement is not a replacement for careful decoding. It is especially promising for Set Cover, while MIS remains challenging because early selection mistakes can irreversibly remove useful vertices.
