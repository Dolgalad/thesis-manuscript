From likelihood to surrogate loss
\[ p_{\hat\theta}(x \mid P) = \frac{1}{Z(\hat\theta)} \exp\!\left( \langle \hat\theta(P;\omega), \phi(x) \rangle \right) \]
\[ \mathcal{L}_{\mathrm{NLL}}(\omega) = -\log p_{\hat\theta(P;\omega)}(y) = \log Z(\hat\theta) - \langle \hat\theta, \phi(y) \rangle \]
The problem: evaluating \(Z(\hat\theta)\) requires summing over all binary assignments.
\[ \mathcal{L}^{(i)}_{\mathrm{LBP}}(\omega) = - \frac{1}{|\mathcal{F}_i|} \sum_{\alpha \in \mathcal{F}_i} \log \mu^{(i)}_{\alpha}(y_{i,\alpha}) \]
LBP returns factor beliefs, so each factor becomes a local multi-class prediction over its configurations.
\[ \mathcal{L}^{(i)}_{\mathrm{NMF}}(\omega) = - \frac{1}{|\mathcal{V}_i|} \sum_{v \in \mathcal{V}_i} \log \mu^{(i)}_{v}(y_{i,v}) \]
This is not exact maximum likelihood: it trains local approximate marginals to match the reference solution.
Key idea: avoid global normalization → fit the MRF through approximate marginal inference
This slide transitions from standard supervised loss to fitting learned MRFs. The natural probabilistic objective is the negative log-likelihood of the reference solution under the predicted MRF. But evaluating this likelihood requires the log-partition function, which is intractable for the loopy factor graphs used here. Instead, Chapter 4 uses supervised marginal-likelihood surrogates. With LBP, the model returns approximate factor marginals, so the loss averages the negative log-probability of the reference local factor configuration. With NMF, the approximation is fully factorized and only variable marginals are available, so the loss becomes a variable-wise surrogate. The important message is that the network is trained through the approximate inference procedure, not by exact maximum likelihood of the full MRF.
