Main idea xt and t provide partial information about x0. Instead of correcting variables independently, the model predicts an MRF and uses marginal inference to propagate structure before denoising.
This slide answers the method question for contribution 2. Start from the problem P, a corrupted assignment x_t, and the diffusion time t. The denoising network predicts the parameters of an MRF for this particular corrupted state. Then BP or naive mean field computes marginals. These marginals are used to estimate the clean assignment x_0, or more generally a denoising distribution. The key oral message is that x_t and t are partial information about x_0. The model should not correct variables in isolation: MRF inference lets information propagate through the structure of the problem before producing the denoising prediction.
